Skincare Analyzer - Setup Guide

Follow these steps to configure your skincare product analyzer with WooCommerce.

üìã Pre-Setup Checklist

  • WordPress website with WooCommerce installed
  • Node.js installed (v14 or higher)
  • Access to WordPress admin panel
  • Products added to WooCommerce store
  • Server with port 3000 available

üîß Step 1: Backend Configuration

1 Navigate to backend directory
cd /home/vibecode/workspace/backend
2 Edit the .env file with your WooCommerce credentials
WOOCOMMERCE_URL=https://your-store.com
WOOCOMMERCE_CONSUMER_KEY=ck_your_key_here
WOOCOMMERCE_CONSUMER_SECRET=cs_your_secret_here
3 Get WooCommerce API credentials:
  1. Login to WordPress admin
  2. Go to WooCommerce > Settings > Advanced > REST API
  3. Click "Add Key"
  4. Set permissions to "Read/Write"
  5. Copy the Consumer Key and Consumer Secret
4 Add OpenAI API Key (Optional but Recommended):

For AI-powered skin image analysis, add your OpenAI API key to the .env file:

  1. Go to OpenAI Platform
  2. Sign up or log in to your account
  3. Click "Create new secret key"
  4. Copy the API key (starts with sk-)
  5. Add to .env file: OPENAI_API_KEY=sk-your-key-here
OPENAI_API_KEY=sk-your-openai-api-key-here

Note: Without OpenAI API key, the system will still work but won't perform AI image analysis. It will use only user-selected conditions for matching.

üè™ Step 2: WooCommerce Product Setup

1 Add ingredients to products

For each product, add a custom field named "ingredients" with comma-separated values:

Example: hyaluronic acid, niacinamide, glycerin, vitamin c
2 Create product categories

Recommended categories: Cleanser, Moisturizer, Serum, Toner, Treatment, Acne Treatment, Anti-Aging

3 Set product prices

Ensure all products have prices set for budget filtering to work

cd /home/vibecode/workspace/backend

2 After updating .env, start/restart the server
